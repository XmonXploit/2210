ROLE DISCORD

Role adalah salah satu keunggulan fitur pada server yang dimiliki oleh discord . Setiap anggota pada server dapat diatur Peran masing masing . Semisal menambahkan admin maka pengaturan role harus di setting pada server discord . Role terletak pada Server Setting - User Management . Setelah Role di buat , member dapat diatur pada role tersebut melalui Server Setting - Member . Namun banyak yang tidak memahami fungsi role ini untuk mengatur anggota di server nya . Nah karena itu Cemiti.id akan memberikan penjelasan pada setiap pengaturan Role pada Server Discord.
