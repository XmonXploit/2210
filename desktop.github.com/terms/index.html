R3 ENDEAVOUR

Syarat dan Ketentuan Aplikasi Open Source Discord ini ("Persyaratan Aplikasi") adalah perjanjian hukum antara Anda (baik sebagai individu atau atas nama entitas) dan Discord, Inc. tentang penggunaan aplikasi Discord, seperti R3 ENDEAVOR dan dokumentasi terkait ("Perangkat Lunak"). Ketentuan Aplikasi ini berlaku ke versi kode Perangkat Lunak yang dapat dieksekusi. Kode sumber untuk Perangkat lunak tersedia secara terpisah dan gratis secara terbuka perjanjian lisensi perangkat lunak sumber. Jika Anda tidak setuju dengan semua persyaratan dalam Ketentuan Aplikasi ini, jangan mengunduh, menginstal, menggunakan, atau salin Perangkat Lunak.

Connecting to R3 ENDEAVOUR

Jika Anda mengkonfigurasi Perangkat Lunak untuk bekerja dengan satu atau lebih akun di situs Discord.com atau dengan contoh Discord Enterprise, penggunaan Anda atas Perangkat Lunak juga akan diatur sesuai keinginan Anda Persyaratan Layanan situs web Discord.com dan / atau perjanjian lisensi berlaku untuk instance ENDEAVOR Enterprise Anda ("Persyaratan R3").

Setiap penggunaan Perangkat Lunak yang melanggar Ketentuan Perselisihan Anda yang berlaku juga merupakan pelanggaran Persyaratan Aplikasi ini.

Licenses and Notices R3 ENDEAVOUR

Lisensi open source untuk Perangkat Lunak disertakan dalam "Open "Pemberitahuan Sumber" yang disertakan dengan Perangkat Lunak.

Sejauh ketentuan lisensi berlaku untuk open source komponen memerlukan GitHub untuk membuat penawaran untuk menyediakan kode sumber sehubungan dengan Perangkat Lunak, tawaran tersebut dibuat dengan ini, dan Anda dapat melakukannya latihlah dengan Discord : https://discord.com/contact.

Kecuali jika disetujui secara tertulis dengan GitHub, perjanjian Anda dengan Discord akan selalu menyertakan, setidaknya, Aplikasi ini Persyaratan. Sampai batas tertentu itu lisensi perangkat lunak sumber terbuka secara tegas menggantikannya Persyaratan Aplikasi, lisensi open source mengatur perjanjian Anda dengan Discord untuk penggunaan Perangkat Lunak atau disertakan secara spesifik komponen Perangkat Lunak.

R3 ENDEAVOUR's Logos

Pemberian lisensi yang disertakan dengan Server bukan untuk Discord's merek dagang, yang menyertakan desain logo Perangkat Lunak. Cadangan perselisihan semua merek dagang dan hak cipta di dan untuk semua merek dagang R3. Logo R3 termasuk, misalnya, desain bergaya itu sertakan "logo" di judul file di folder "logo".

Nama R3 ENDEAVOR, dan logo Endeavour terkait dan / atau bergaya nama adalah merek dagang dari R3. Anda setuju untuk tidak menampilkan atau menggunakan merek dagang ini dengan cara apapun tanpa perselisihan sebelumnya, tertulis izin, kecuali sebagaimana diizinkan oleh Logo R3 ENDEAVOUR dan Kebijakan Penggunaan: https://endeavour.com/logos.

Privacy R3

Perangkat Lunak dapat mengumpulkan informasi pribadi. Anda dapat mengontrol apa informasi yang dikumpulkan Perangkat Lunak di panel pengaturan. Jika Perangkat lunak memang mengumpulkan informasi pribadi atas nama R3 ENDEAVOUR, R3 akan memproses informasi tersebut sesuai dengan R3 Privacy Statement.

Additional Services

Auto-Update Services

Perangkat Lunak dapat mencakup layanan pembaruan otomatis ("Layanan"). Jika kamu memilih untuk menggunakan Layanan atau Anda mengunduh Perangkat Lunak itu secara otomatis mengaktifkan Layanan, R3 akan diperbarui secara otomatis Perangkat Lunak saat versi baru tersedia.
